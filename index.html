Aplikasi Game 2D Platformer "Dragon Warrior" Pada Windows

Dragon Warrior adalah game 2D platformer yang saya kembangkan menggunakan Unity dan C#, terinspirasi dari Super Mario Bros. Game ini berfokus pada rintangan (obstacle-based) di mana pemain harus melompati hambatan, menjaga health, dan mencapai garis finish. Pengembangan game dilakukan menggunakan metode MDLC, mulai dari perancangan, pembuatan aset, hingga pengujian Blackbox Testing dan User Acceptance Testing (UAT).

Game ini memiliki tiga level dengan tingkat kesulitan berbeda, dilengkapi karakter 2D, musuh, rintangan, efek audio, serta UI sederhana. Dragon Warrior berjalan pada platform Windows dan dirancang untuk memberikan pengalaman platformer klasik dengan gameplay menantang dan timing lompatan yang presisi

Website Jasa "Toko Parfum Laundry"

Website Toko Parfum Laundry adalah aplikasi web layanan laundry yang saya kembangkan menggunakan PHP dan MySQL. Sistem ini memungkinkan pelanggan melakukan pemesanan layanan secara digital melalui fitur katalog, pencarian layanan, dan form pemesanan online. Di sisi admin, aplikasi ini menyediakan dashboard dengan login aman, serta fitur CRUD untuk mengelola layanan, produk parfum, dan pesanan pelanggan. Alur pemesanan dirancang terstruktur, mulai dari permintaan pelanggan hingga proses persetujuan admin, sehingga operasional menjadi lebih efisien.

Proyek ini berhasil lolos uji kompetensi BNSP, di mana seluruh fitur dinilai memenuhi standar nasional dan diuji melalui praktik langsung.

Fitur Utama:
  • Manajemen layanan & produk
  • Form pemesanan pelanggan
  • Halaman admin

Aplikasi Game 2D Platformer "Dragon Warrior: Arise" Pada Android

Game 2D platformer berbasis Android yang saya kembangkan menggunakan Unity. Proyek ini merupakan pengembangan dari versi PC sebelumnya dengan peningkatan navigasi level, kontrol permainan, serta penambahan total 6 level baru.
